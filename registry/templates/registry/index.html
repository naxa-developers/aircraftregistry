Aircraft Registry
What is this?

Welcome to the Aircraft registry sandbox. This site has three things:

  • Registry Landscape Whitepaper
  • Interoperatble API Specification, the technical specification for a registry. You can see the API specification and explore API endpoints by reviewing API Blueprint
  • A working API with all endpoints and sample data for you to explore, more below.

Sandbox Data

The sandbox registry is populated by dummy / test data, take a look at it here. You can create a JSON file with you own data to test out the different functions.

Implemented Endpoints

Explore the implemented endpoints using the sandbox data by clicking on links below:

  • /api/v1/operators - Operator List
  • /api/v1/contacts - Contact List
  • /api/v1/operator/{operatorid} - Details about a single operator
  • /api/v1/operator/{operatorid}/privilaged - Privilaged details about a single operator
To review a full list of calls, please review the blueprint specifications.
Contributing
